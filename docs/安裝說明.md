# Quill 安裝說明

嗨!感謝你幫忙試用 Quill。

**Quill 是什麼**:在 Mac 上框選螢幕任何區域,就能直接問 AI——翻譯、擷取文字、解釋內容,結果當場浮現在旁邊,不用切去 ChatGPT。也能選取文字直接改寫(修錯字、改語氣、翻譯)。

安裝大約 3 分鐘。因為還在測試階段、尚未上架,所以需要多幾個步驟,以下每一步都有寫清楚。

---

## 步驟 1:安裝

1. 下載我傳給你的 `Quill.zip`,雙擊解壓縮
2. 把解壓出來的 **Quill** 拖進「**應用程式**」資料夾

## 步驟 2:第一次打開(重要!)

因為這個版本還沒經過 Apple 的公證程序,**不能直接雙擊打開**,要用這個方式:

1. 打開「應用程式」資料夾,找到 Quill
2. **在 Quill 上按右鍵**(或按住 Control 點一下)→ 選「**打開**」
3. 跳出警告視窗後,再點一次「**打開**」

> 如果還是打不開,或根本沒有「打開」選項:
> 到「**系統設定**」→「**隱私權與安全性**」→ 往下捲動,會看到一行「已阻擋使用 Quill」→ 點旁邊的「**仍要打開**」,然後回到步驟 2 再試一次。

這個警告是正常的,只是因為軟體還沒上架。之後正式版就不會有了。

## 步驟 3:跟著引導開權限

打開後,Quill 會出現在螢幕**最上方的選單列**(找羽毛筆圖示),並跳出設定引導視窗。跟著它走就好,總共兩個權限:

**輔助使用**——讓 Quill 能讀取你選取的文字。點引導裡的按鈕會直接開啟設定頁,找到 Quill 打開開關即可。

> 如果系統設定的清單裡**找不到 Quill**:回到引導視窗,點「清單裡沒有 Quill?複製 App 路徑」,然後在系統設定清單下方點「**+**」,按 `Cmd + Shift + G`,貼上路徑,選 Quill 加入。

**螢幕錄製**——截圖功能需要。勾選後**一定要重新啟動 Quill 才會生效**,引導視窗上會有「重新啟動」按鈕,點它就好(重啟後會自動回到原本的步驟)。

## 步驟 4:開始用

引導走完就能用了,不需要註冊、也不需要任何 API key。

- **截圖問 AI**:按 `Control + Option + Q`,拖曳框選畫面任何區域,選一個動作(翻譯 / 擷取文字 / 解釋…),結果會浮現在旁邊,還可以在視窗裡繼續追問
- **選字改文字**:在任何輸入框選取文字,按 `Control + Option + A`,選動作後會直接取代原文

每天有 **10 次免費額度**,每日重置。

## 常見狀況

**按快捷鍵沒反應**
多半是「輔助使用」權限沒生效。點選單列的羽毛筆圖示 →「設定引導」,回去確認權限那頁是不是綠色勾勾。

**截圖拍到的畫面沒有視窗內容(只有桌布)**
「螢幕錄製」權限沒生效,或勾選後還沒重新啟動 Quill。到設定引導的螢幕錄製那頁,點「重新啟動」。

**顯示「今日免費額度已用完」**
一天 10 次的額度用完了,明天會重置。

**想換快捷鍵**
選單列羽毛筆圖示 →「Preferences」→ 可以自訂兩個快捷鍵。
