# 打包 Quill 給朋友(未簽章測試版)

還沒辦 Apple Developer 前,用這個方式把 App 傳給朋友試用。約 5 分鐘。

## 一、從 Xcode 匯出 App

1. Xcode 選單 **Product → Show Build Folder in Finder**
2. 進 `Products/Debug/`,找到 **Quill.app**
3. 在 Quill.app 上按右鍵 → **壓縮**,得到 `Quill.zip`

## 二、兩種傳給朋友的方式

**方式 A:私下傳(最快)**
把 `Quill.zip` + `docs/安裝說明.md` 一起用 Google Drive / AirDrop / LINE 傳給朋友。

**方式 B:放上官網下載(給更多人)**
官網的「免費下載」按鈕已指向同目錄的 `Quill.zip`。做法:
1. 把 `Quill.zip` 放進 `landing/` 資料夾
2. `git add landing/Quill.zip && git commit && git push`
3. GitHub Pages 部署後,官網點「免費下載」就會下載到這個 zip

> 注意:GitHub 單檔上限 100MB,Quill.zip 應遠小於此。每次改版重新覆蓋 `landing/Quill.zip` 再 push 即可。

## 三、注意

- 未簽章 = 朋友第一次開會看到「無法驗證開發者」,務必附上 `安裝說明.md`。
- App 內含你的 Cloud 密鑰,朋友用的是你的每日免費額度 + Railway 後端(GLOBAL_DAILY_CAP=400 已保護成本)。
- 未簽章版沒有自動更新;每次改版都要手動重傳。辦好 Apple Developer、出簽章 DMG 後,自動更新才會生效。
