常見問題

常見問題，一次說清。

需要 OpenAI 帳號嗎？

不需要——Quill 支援任何 OpenAI 相容的服務商。可選 OpenAI、Groq、OpenRouter，或在本地跑 Ollama（完全免費）。Quill Cloud（即將推出）內含點數，下載即用，不需任何 API Key。

為什麼 Quill 需要「輔助使用」權限？

這是 Quill 讀取選取文字並在原位取代的必要權限。只需在「系統設定」授權一次，相關程式碼都開放在 GitHub 供查核。

哪些 App 可以用？

幾乎所有 App 都支援——Quill 使用 macOS 輔助使用 API，並提供剪貼板備援方案：

App 類型讀取選取文字原位改寫
TextEdit、備忘錄、原生 App完整支援
Safari、Preview（PDF）完整結果面板
Chrome、Electron App剪貼板備援結果面板

可以用其他 AI 模型嗎？

可以——在偏好設定選擇供應商與模型名稱。內建預設涵蓋 OpenAI（gpt-4o-mini）、Groq（llama-3.3-70b-versatile）、OpenRouter（openai/gpt-4o-mini）及 Ollama（llama3.2），任何 OpenAI 相容端點都支援。

標題的 demo 有真的呼叫 API 嗎？

沒有——這個頁面的互動是腳本模擬，用來呈現真實的操作流程。真正的 App 會將你的選取內容送到你設定的模型。
